2.5. Add flow charts in JB#

First, install the Sphinx extensions

pip install sphinxcontrib-mermaid
pip install sphinxcontrib-wavedrom

Then, in the _config.yml file, add

sphinx:
  extra_extensions:
    - sphinxcontrib.mermaid
    - sphinxcontrib.wavedrom

2.5.1. Mermaid flow chart#

You can start to use the mermaid systax to make your own flow chart :)

2.5.1.1. Top down flow chart#

```{mermaid}

flowchart TD
    A[Start] --> B{Is it?}
    B -->|Yes| C[OK]
    C --> D[Rethink]
    D --> B
    B ---->|No| E[End]
```

The result is

flowchart TD A[Start] --> B{Is it?} B -->|Yes| C[OK] C --> D[Rethink] D --> B B ---->|No| E[End] 
```{mermaid}
flowchart TD
    
    A:::someclass --> B:::someclass
    classDef someclass fill:#f96,stroke:#333;

```
flowchart TD A:::someclass --> B:::someclass classDef someclass fill:#f96,stroke:#333;

2.5.1.2. Left-Right flow chart#

```{mermaid}
flowchart LR
    A[Start] --> B{Is it?}
    B -->|Yes| C[OK]
    C --> D[Rethink]
    D --> B
    B ---->|No| E[End]
```

The result is

flowchart LR A[Start] --> B{Is it?} B -->|Yes| C[OK] C --> D[Rethink] D --> B B ---->|No| E[End]
